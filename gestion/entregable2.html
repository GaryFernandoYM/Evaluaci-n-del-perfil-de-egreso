2. Catálogo de Procesos

PRO-01 · Registro facial
Actores: Sistema (cámara + IA), Docente, BD
Disparador: Inicio de clase / hora de entrada
Resultado: Registro automático + notificación
PRO-02 · Generación de reportes
Actores: Administrador, Director, Sistema web
Disparador: Solicitud de reporte diario/semanal/mensual
Resultado: PDF/Excel con asistencias y faltas
PRO-03 · Gestión de personas
Actores: Administrador
Disparador: Matrícula o cambio de personal
Resultado: Persona registrada con foto para IA
PRO-04 · Entrenamiento del modelo
Actores: Sistema IA (FastAPI), Administrador
Disparador: Nuevas fotos cargadas
Resultado: Modelo actualizado, mayor precisión
PRO-05 · Notificaciones a padres
Actores: Sistema, Firebase FCM
Disparador: Registro de falta o tardanza
Resultado: Notificación push al apoderado

3. Diagrama de Flujo — PRO-01 (Registro Facial)

Diagrama BPMN proceso PRO-01 · gestion-e2-bpmn-pro01.png
1
Alumno/docente se presenta ante la cámara
2
Sistema captura frame de video
3
FastAPI detecta y codifica el rostro
4
Sistema compara con embeddings registrados
5a
Si similitud > 70%: registra asistencia en MySQL
5b
Si no coincide: registra como "desconocido", alerta al docente
6
Laravel muestra el registro en tiempo real en el dashboard
