3. Proceso de Despliegue

1
Desarrollador hace commit y push a rama main en GitHub
2
GitHub Actions ejecuta pipeline — si falla, el proceso se detiene
3
Acceder al VPS por SSH
4
git pull origin main
5
composer install --no-dev --optimize-autoloader
6
php artisan migrate --force
7
php artisan config:cache && php artisan route:cache
8
sudo systemctl restart php8.3-fpm nginx
9
Verificar en el navegador que el sistema responde correctamente
