2. Descripción del Sistema

QhawAI es un sistema de control de asistencia con reconocimiento facial. Su infraestructura es distribuida y orientada a servicios, sin depender de una red LAN física propia — aprovecha servicios en la nube (Firebase) y un servidor VPS.

4. Requerimientos de Infraestructura

RI-01
VPS con mínimo 2 vCPU y 2 GB RAM para ejecutar Laravel + FastAPI simultáneamente.
RI-02
PHP >= 8.3, Python >= 3.10, MySQL >= 8.0 instalados en el servidor.
RI-03
Comunicaciones cliente-servidor mediante HTTPS (TLS 1.2+) para proteger datos biométricos.
RI-04
Firebase configurado con reglas de seguridad que restrinjan acceso a usuarios autenticados.
RI-05
Sistema soporta hasta [FALTA] usuarios concurrentes sin degradación.
RI-06
Tiempo de respuesta de la API de reconocimiento facial <= 3 segundos.
RI-07
Base de datos con backup automático diario.
