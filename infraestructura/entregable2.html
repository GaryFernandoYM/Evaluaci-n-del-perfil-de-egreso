2. Configuración del Servidor Web (Nginx)

Nginx actúa como proxy reverso, recibiendo peticiones HTTPS y redirigiendo a Laravel via PHP-FPM en socket Unix.

server {
    listen 443 ssl;
    root /var/www/qhawai/public;
    index index.php;
    ssl_certificate /etc/letsencrypt/live/[dominio]/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/[dominio]/privkey.pem;

    location ~ \.php$ {
        fastcgi_pass unix:/run/php/php8.3-fpm.sock;
    }
}

4. Configuración de FastAPI (Motor IA) — Docker + Nginx

FastAPI se ejecuta dentro de un contenedor Docker con Nginx como servidor web interno. Docker gestiona el ciclo de vida del contenedor automáticamente.

ParámetroValor
TecnologíaDocker (contenedor)
Servidor web internoNginx (dentro del contenedor)
Puerto interno80 (solo acceso desde el host VPS)
RuntimeUvicorn + FastAPI (Python 3.11)
InicioAutomático con Docker (restart: always)
ℹ️
Flujo interno del contenedor: Nginx :80 → proxy → Uvicorn :8000 → FastAPI Python
Laravel se comunica con el contenedor vía http://localhost:80 
docker ps                          # ver contenedor corriendo
docker logs [nombre-contenedor]    # ver logs recientes
docker restart [nombre-contenedor] # reiniciar si es necesario
